# Final Project (Sports Analytics Course - 36-460)

## Overview
This project investigates how MLS teams allocate salary under strict budget constraints to achieve on-field success. We explore whether competitive advantage is driven by total payroll, spending concentration, or positional allocation.

## Research Question
How do salary allocation and roster construction relate to team performance in MLS?

## Individual Analyses

### 1. Spending Efficiency
Evaluates if performance increases with total payroll and how efficiency—converting salary into production—affects results.

### 2. Roster Architecture
Examines whether success depends on salary disparity (inequality) or the depth of the "middle class" (TAM-eligible players).
